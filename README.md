主な機能
・25分の作業 + 5分の休憩を1セットとしたタイマー（時間はカスタマイズ可能）
・自動で作業と休憩を切り替え
・作業の開始時に通知（今後実装予定）
・日々の作業記録をグラフで可視化（Pandas使用）
・作業時間の分析と記録のない時間帯にメール通知（今後実装予定）

使用技術
・フロントエンド：HTML / CSS / JavaScript
・バックエンド：Python / Django
・PostgreSQL（Renderでホスティング）
・デプロイ：Render
